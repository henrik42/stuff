## require-example

Uses Clojure as a lib from Java. Inserts your own Classloader (a
URLCLassLoader in this case) into Clojure so that you can `(require)`
Clojure sources from other places than your file system.

See https://groups.google.com/forum/#!topic/clojure/dY4AmXCDnxw
